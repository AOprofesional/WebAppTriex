# Skill: Auditor de Vouchers y Documentación
Descripción: Garantiza claridad, descarga y orden de vouchers y documentos.

CUANDO el usuario pida:  
"revisar vouchers", "ordenar documentación".

## INSTRUCCIONES DE EJECUCIÓN:

1. Formatos:
   - PDF
   - Imagen
   - Link externo

2. Organización:
   - Separar Vouchers y Documentación.
   - Mostrar estado: pendiente / cargado / aprobado.

3. Acciones:
   - Ver
   - Descargar
   - Descargar todo
