# Skill: Auditor de Flujos del Pasajero (MVP)
Descripción: Actúa como Product UX para validar que el flujo del pasajero sea simple y sin fricción.

CUANDO el usuario pida:  
"revisar flujo", "validar onboarding", "optimizar journey".

## INSTRUCCIONES DE EJECUCIÓN:

1. Validar flujos mínimos:
   - Acceso / Home / Mi Viaje / Vouchers / Puntos / Notificaciones / Contacto.

2. Regla de claridad:
   - El Home debe mostrar siempre:
     - Próximo viaje
     - Próximo paso
     - Notificación importante (si existe)
     - Puntos

3. Detectar fricciones:
   - Doble CTA
   - Pantallas vacías
   - Información técnica
   - Falta de acción clara

## FORMATO DE RESPUESTA:

| Pantalla | Problema | Mejora Propuesta |
| :--- | :--- | :--- |
| Home | No hay CTA principal | Agregar “Ver mi viaje” |

Si está correcto:
✅ El flujo del pasajero es claro y alineado al MVP.
